vubat

system tray ibam frontend

vubat displays the status, adapted battery timing and percentage on mouse hover. every 2 seconds ibam or ACPI is called in read-only mode to get current data. When using ibam, every 2 minutes the update process calls for a sample to be stored in ibam.

    icon colors:
  • blue = chargin, charged
  • green = 100% - 60%
  • yellow = 60% - 30%
  • orange = 30% - 10%
  • red = 10% - 0%
    prerequisites:
  • python
  • python-gtk2, python-gobject
  • ibam or acpi
